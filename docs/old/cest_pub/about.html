◆ Consortium for Embedded Software Technology
    『組込みソフトウェア開発技術研究会（CEST）』は、 愛知県三河地区を中心とした産学共同の研究・開発活動を行う団体です。
    CESTは、活動を通じて会員の組込みソフトウェア開発技術を向上させること、 また会員企業の産業分野における競争力の向上を目指しています。

◆ CESTへの参加方法
以下に簡単な流れを説明いたします。

  1. CEST会員規約に同意ください。
  2. 郵便または電子メイルで入会申し込みをお送りください。
  3. 参加の可・不可を折り返しご連絡します。
    （基本的に可ですのでご気軽に）
  4. CESTメイリングリストで参加者のご紹介を行います。
参加を希望される方は、下記の参加案内をご利用ください。
