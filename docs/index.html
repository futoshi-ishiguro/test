What's CEST?

「組込みシステム開発技術研究会（Consortium for Embedded System Technology; CEST）」は、東海地方を中心とし た産学共同の研究・開発活動を行う団体です．活動を通じて会員の組込み システム開発技術を向上させること、また会員企業の産業分野における競争力の向上を目指しています．

第246回技術交流会

  • 日時：2021年03月17日（水）18:30-21:00
  • 場所： ウインクあいち1208会議室（場所）またはオンライン
  • 話題提供者：山路厚氏、西村隆氏（デンソークリエイト）
  • 話題：CASE時代に求められる”デザインDX”　「トレーサビリティの事」
  • 概要：CASE時代により、大規模で複雑なシステムをコンカレントに開発することが現場 には求められています。 その中においても、より素早く高品質に開発を進めるために「デザインDX」、つ まり、設計情報のデジタル化が必要であると考えています。 設計情報のデジタル化の効果は様々ですが、今回はトレーサビリティを事例とし てご紹介します。 本取り組みに関して、皆さまと議論させてください。
  • 締め切り：3月15日（月）
