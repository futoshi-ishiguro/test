What's CEST?

「組込みシステム開発技術研究会（Consortium for Embedded System Technology; CEST）」は、東海地方を中心とし た産学共同の研究・開発活動を行う団体です．活動を通じて会員の組込み システム開発技術を向上させること、また会員企業の産業分野における競争力の向上を目指しています．
