「組み込みソフトウェアに必要な試験、品質管理、品質保証」

中国での人工衛星打ち上げ成功と、日本の人口衛星打ち上げ失敗は、 日本の技術の評価に大きな打撃を与えています。

人工衛星のように、利用するものを直接試験できないような場合を 含めて、組み込みソフト開発における試験のあり方は重要性を増して います。

ITRON, Linux, Windows CEをはじめ、組み込みソフトウェアに関す る技術的な発展はめざましいものがあります。しかし、現場で必要と なる初歩的な試験が省略されることによる不具合が頻発していないで しょうか。品質管理、品質の保証のための試験は、開発の上流から下 流に至るすべての工程において必要になっています。

そこで、最近の技術動向を背景として、実際に必要となる試験につ いての知見を紹介し、 契約相手から取得するべき試験結果をはじめ、 全方位的な対応について議論できればと考えています。
